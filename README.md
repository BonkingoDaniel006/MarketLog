MarketLog est une application de gestion de stock simple, intuitive et robuste, développée en Python avec Tkinter.
Elle permet de gérer efficacement un petit inventaire de produits alimentaires grâce à une interface graphique claire et des fonctionnalités essentielles de gestion commerciale.
L’application repose sur une architecture multi‑écrans propre et modulaire, ce qui la rend facile à maintenir, à étendre et à comprendre.

 Fonctionnalités principales
 
1. Gestion du stock en temps réel
MarketLog maintient automatiquement un inventaire à jour.
Chaque action (vente, ajout, réapprovisionnement) modifie instantanément les quantités disponibles.
2. Vente de produits
- Saisie du code produit
- Vérification automatique de l’existence du produit
- Contrôle du stock disponible
- Décrémentation automatique du stock
- Mise à jour immédiate dans la base de données CSV
3. Réapprovisionnement
- Ajout d’un nouveau produit via un formulaire en plusieurs étapes
- Mise à jour du stock d’un produit existant
- Enregistrement automatique dans le fichier CSV
4. Consultation du stock
- Affichage dynamique de tous les produits
- Le tableau se met à jour automatiquement à chaque ouverture
- Informations affichées : code, nom, prix, stock
5. Consultation du prix d’un produit
- Recherche par code
- Affichage clair du nom et du prix du produi
